# Product

## Register

product

## Users

Developers, operations engineers, platform engineers, and other technical users who manage object storage during daily work. They are usually checking buckets, moving files, validating uploads, sharing temporary links, or inspecting object metadata while staying in a focused desktop workflow.

## Product Purpose

BucketDesk is a desktop object storage manager for S3-compatible services, MinIO, Aliyun OSS, and Tencent COS. It helps users connect to multiple providers, browse buckets and objects, upload and download files, preview supported formats, manage folders, and perform copy, move, rename, delete, and presigned-link operations with clear feedback and low friction.

## Brand Personality

Practical, precise, calm. The interface should feel like a professional file manager: predictable, dense enough for repeated work, and confident without visual noise.

## Anti-references

Avoid marketing-style dashboards, oversized hero panels, decorative card layouts, and cloud-console clutter that hides common file operations. Avoid designs where preview, open, download, and metadata inspection feel like the same action.

## Design Principles

1. File management first: bucket and object browsing should behave like a familiar desktop file manager.
2. Separate action intent: open folders, preview files, inspect details, download, and destructive actions must have distinct affordances.
3. Preserve professional density: show enough metadata and controls for expert users without burying the primary path.
4. Make state visible: loading, selection, active path, transfer start, unsupported preview, and errors should be explicit.
5. Keyboard and accessibility matter: core browsing and actions should be reachable without a mouse, with clear focus and contrast.

## Accessibility & Inclusion

Target WCAG AA for contrast, keyboard access, focus indicators, semantic controls, and reduced-motion support. The interface should work for users who rely on keyboard navigation, need clear text labels, or benefit from color-independent status indicators.
